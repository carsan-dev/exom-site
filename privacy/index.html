Privacidad

Datos necesarios para prestar el servicio.

EXOM se usa con cuentas creadas por el entrenador. Esta política explica qué datos se tratan y cómo pedir ayuda.

Responsable y contacto

El servicio EXOM Method atiende solicitudes de privacidad en exom.method@gmail.com.

Datos tratados

EXOM puede tratar datos de cuenta, identificadores técnicos, información de entrenamiento, nutrición, progreso, feedback, notificaciones y contenido que el usuario suba dentro de la app.

Finalidad

Los datos se usan para gestionar el acceso, entregar planes de entrenamiento y nutrición, medir el progreso, enviar notificaciones y prestar soporte.

Acceso limitado

No existe auto-registro público. Las cuentas las crea o autoriza el equipo responsable del servicio.

Proveedores técnicos

EXOM utiliza servicios de infraestructura, autenticación, almacenamiento, notificaciones y analítica técnica necesarios para operar la plataforma.

Derechos

Puedes solicitar acceso, rectificación, eliminación o revisión de tus datos escribiendo a exom.method@gmail.com. También puedes iniciar la eliminación desde eliminar datos.
